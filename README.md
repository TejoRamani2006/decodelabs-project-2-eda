# Project 2 - Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to analyze the dataset and identify patterns, trends, and outliers using exploratory data analysis techniques.

## Tools Used

* Python
* Pandas
* Matplotlib
* VS Code

## Analysis Performed

1. Examined dataset structure and columns.
2. Calculated basic statistics such as mean, median, and count.
3. Analyzed product distribution.
4. Analyzed order status distribution.
5. Detected outliers using the IQR method.
6. Created visualizations for better understanding of the data.

## Key Findings

* Printer was the most frequently ordered product.
* Average order value was approximately ₹1054.
* A small number of outliers were identified in TotalPrice.
* Product sales were distributed across multiple categories.
